# Platform Components

This directory is reserved for Kubernetes platform components managed by GitOps.

Expected future contents:

- Ingress controller.
- cert-manager.
- External Secrets or Vault Secrets Operator.
- Observability collectors and agents.
- Runtime security tooling.
- Admission policy controllers.
- Network policy baseline.
- Backup and restore operators.

The platform layer should be installed before hosted applications are promoted to
Meridian.
