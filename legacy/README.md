# Legacy Scope

This directory contains earlier MERIDIAN platform concepts that are no longer part
of the active v2 scope.

MERIDIAN v2 is being narrowed to Kubernetes runtime security and detection
engineering. The material here is retained for reference only and should not be
read as implemented functionality.

Archived themes:

- broad homelab platform design
- selective cloud integration planning
- static site hosting
- GitOps and IaC placeholders
- AWS Vault configuration placeholder

These ideas may be revisited later, but they are intentionally outside the active
MERIDIAN v2 detection-engineering path.
